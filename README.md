# Bioinformatics Lab Analysis

Projeto fictício para prática de análise de dados laboratoriais e bioinformática utilizando Python.

## Tecnologias
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Estrutura
- data/: datasets
- notebooks/: análises
- images/: gráficos

## Objetivo
Explorar dados laboratoriais simulados e gerar insights biológicos.
